I'm Godspower Essien, a self-taught software developer from Uyo, Akwa Ibom, Nigeria. Since 2024, I've been building software with growing confidence — learning fast, shipping real things, and treating every project as a lesson.

My stack spans Python and Django for backends, React for frontends, and Flutter for mobile. I believe in learning by building — each line of code teaches me something, and I ship things that actually work in the real world.
